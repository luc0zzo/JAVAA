gli if nel linguaggio JAVA


cos'è un if?

L'istruzione "if" in Java è fondamentale per il controllo del flusso di un programma. Consente di eseguire un blocco di codice solo se una determinata condizione è vera. La sintassi di base è "if (condizione) { blocco di codice }". Si può anche utilizzare "else" per specificare un blocco di codice da eseguire se la condizione non è vera. Inoltre, è possibile concatenare più istruzioni "if" utilizzando "else if" per controllare più condizioni in modo sequenziale. L'istruzione "if" è ampiamente utilizzata per creare logica decisionale nei programmi Java, consentendo di gestire vari scenari e di eseguire codice in base alle condizioni del programma.


if: condizione

in java si visualizza nel seguente modo: 
            
            if: 
            if(condizione booleana){
                azione
            }



condizione if/else

La condizione if/else è una struttura di controllo in programmazione che consente di eseguire blocchi di codice diversi in base al verificarsi di una condizione booleana. Se la condizione è vera, viene eseguito il blocco di codice sotto if, altrimenti viene eseguito il blocco di codice sotto else. 

 
            if/else :
            if(condizione booleana){
                azione
            }
            else(condizione booleana){
                azione
            }



condizione if/else/else if

La struttura if/else if/else è una serie di istruzioni condizionali utilizzate per eseguire blocchi di codice diversi in base a più condizioni. Il flusso del programma procede dall'alto verso il basso e viene eseguito il primo blocco di codice il cui test di condizione è vero. Se nessuna delle condizioni è vera, viene eseguito il blocco di codice nell'istruzione else, se presente. 

        
            if/else if/else: 
            if(condizione booleana){
                azione
            } 

            else if(condizione booleana){
                azione
            }

            else{
                azione
            }



lo switch

Lo statement switch è una struttura di controllo in programmazione utilizzata per selezionare un ramo di esecuzione da un numero fisso di possibilità basate sul valore di un'espressione.
Quando viene eseguita, l'espressione viene valutata una volta e il controllo passa al primo caso corrispondente. Se nessun caso corrisponde, il codice nel blocco default (facoltativo) viene eseguito. L'istruzione break è utilizzata per interrompere l'esecuzione del switch dopo che il caso corrispondente è stato trovato ed eseguito. 

            
                switch: 
                switch(espressione) {
                    case valore1:
                        // blocco di codice se espressione è uguale a valore1

                        break;
                    case valore2:
                        // blocco di codice se espressione è uguale a valore2

                        break;
                    // altri casi possibili...
                    default:
                        // blocco di codice se nessun caso corrisponde a espressione

                }
